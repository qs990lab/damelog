# コンポーネント依存関係

## 依存関係マトリクス

| コンポーネント | 依存先 |
|--------------|--------|
| C1 (フロントエンド) | C2, C8, C9 |
| C2 (エージェント) | C3, C4, C5, C6, C10, Memory, Bedrock LLM |
| C3 (進化判定) | C7 |
| C4 (技解放) | C7, Bedrock LLM |
| C5 (モンスター生成) | C7, Bedrock LLM |
| C6 (図鑑) | C7 |
| C7 (DynamoDB) | - |
| C8 (S3) | - |
| C9 (Cognito) | - |
| C10 (Guardrails) | - |

## 通信パターン

- **同期（ストリーミング）**: C1 → C2（HTTP streaming）
- **同期（ツール呼び出し）**: C2 → C3/C4/C5/C6（Gateway MCP）
- **同期（データアクセス）**: C3/C4/C5/C6 → C7（DynamoDB SDK）
- **認証**: C1 → C9（Cognito SDK、リクエスト前に検証）
