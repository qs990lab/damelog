## 通信パターン

- **同期（ストリーミング）**: C1 → C2（HTTP streaming）
- **同期（ツール呼び出し）**: C2 → C3/C4/C6/C11/C14（Gateway MCP）
- **同期（Guardrails）**: C2 → C10（Bedrock Guardrails API）
- **同期（メモリ）**: C2 → C12（AgentCore Memory API）
- **同期（LLM）**: C2 → C13（Bedrock InvokeModel API）
- **同期（データアクセス）**: C3/C4/C6/C11/C14 → C7（DynamoDB SDK）
- **独立実行**: C5 → C7, C8（Python CLI直接実行）
- **認証**: C1 → C9（Cognito SDK、リクエスト前に検証）
