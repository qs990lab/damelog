## 開発フェーズ

| フェーズ | ユニット | 前提条件 |
|---------|---------|---------|
| Phase 0 | U1 | なし |
| Phase 1 | U9, U2, U3, U4, U5 | U1完了 |
| Phase 2 | U6, U7 | U1完了 |
| Phase 3 | U8 | U1完了 |
| Phase 4 | U10 | U1, U6完了 |

※ Phase 1内のU9は他と並行実行可能（独立CLI）
※ Phase 1内のU2はU3に依存するため、U3のAPI仕様確定後にU2のエージェント連携部分を実装する
