# ペルソナ定義

## ペルソナ 1: 田中 翔太（たなか しょうた）— ストレス逃避型会社員

| 項目 | 内容 |
|------|------|
| 年齢 | 29歳 |
| 職業 | IT企業のプロジェクトマネージャー |
| 性格 | 真面目だが完璧主義で疲れやすい。仕事後の「ご褒美」が日課 |
| ダメ行動傾向 | 深夜ラーメン、SNS3時間、二度寝、課金 |
